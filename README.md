Genotet
=======

Genotet: An Interactive Web-based Visual Exploration Framework to Support Validation of Gene Regulatory Networks


Installation:
1) Install node.js on your machine.
2) Put the genotet folder to your server web directory, or your localhost web directory (e.g. xampp).
3) Install required packages in the server folder:
  cd server
  npm install express
  npm install body-parser
4) Modify server/server.js to locate the data at correct directories.
5) Run the server code
node server/server.js
